Sobre o Projeto

Este site é a materialização de um Trabalho de Conclusão de Curso (TCC) em Licenciatura em Física. O projeto foi concebido para oferecer ferramentas tecnológicas que promovam a acessibilidade no ensino de ciências, focando em diminuir as barreiras de comunicação para estudantes surdos.

A plataforma une duas abordagens: um Tradutor Livre, para textos abertos, e um Glossário Terminológico, para conceitos específicos de Física. Ambas as ferramentas utilizam Libras para criar um ambiente de aprendizado mais inclusivo e eficaz.
